# **App Name**: Apex Portfolio

## Core Features:

- Project Gallery: Visually display featured projects with descriptions and images.
- About Me: Showcase personal background, experience, and professional details.
- Skills Display: Illustrate key skills with a visually engaging presentation.
- Contact Form: Enable direct communication via an integrated contact form.
- Dark Mode: Toggle between light and dark themes.
- AI-Powered Content Tool: Use AI to dynamically adjust content presentation, ensuring optimal layout and readability across devices.

## Style Guidelines:

- Primary color: Vibrant emerald green (#26CF78) to highlight a modern and professional look.
- Background color: Dark, desaturated green (#2B3632) for a sophisticated feel that doesn't distract from content. Suitable for dark scheme.
- Accent color: A lighter analogous green (#30D982) for interactive elements and highlights.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines, paired with 'Inter' (sans-serif) for body text.
- Use sleek, minimalist icons to represent skills and project categories.
- Employ a grid-based layout for consistent alignment and responsive design.
- Subtle transition effects on project hover and content loading.
